Maze Game Prototype Instruction Manual

  • Raeciel Reyes
  • Huijun Zhou

Dependencies

  • Phaser.io version 2.3.0

Requirements:

  • Install the emulator.
  • The game uses Phaser.io, and which requires a server to run. The emulator comes with a node server, or if you have python pre-installed, you can use a python's SimpleHTTPServer.

How to play:

  • Select "Maze Prototype" from the drop-down menu, then click "start".
  • Simply use your "Up, Down, Left, Right" key on the keyboard to move the rectangle.
  • The rectangle will not be able to pass the black pixels when they collide.
  • The only way to continue to the next level is reach the black hole in the centre.

Bugs

  • At times, the picture/maze of game does not show up.
    Solution: All you need to do is double click the "start" button.

Reference
