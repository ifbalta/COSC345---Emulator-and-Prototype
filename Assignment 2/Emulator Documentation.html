Emulator Documentation

authors:
  • Isabel Baltazar
  • Lennox Huang

The Emulator is written in JavaScript and jQuery. It consists of a canvas acting as a watch face, and a drop down menu as the app launcher.
The emulator is compatible with the Safari browser.

Features

  • A screen
  • An app tray

For Users:

Starting up the emulator is as simple as opening emulator.html in Safari, and selecting an app from the menu.

For Developers

The emulator's engine is written in JavaScript and makes use of the jQuery version 1.11 sourced from Google's Ajax library.
The emulator relies on the following:

  • bootup()

    Instantiates the appObject.

  • clear()

    Clears the screen.

  • selectApp()

    Linked to the start button to select an app.

Installation

Get the code here.
