The aim of the maze game application is to reach the hole at the centre of the maze.
The ball movement can be controlled by arrow keys. There are multiple levels of increasing difficulty.

Requirements:

  • Install the emulator.
  • The game uses Phaser.io's game engin, which requires a server to run. The emulator comes with a node server, or if you have python pre-installed, you can use python's SimpleHTTPServer.

How to play:

  • Simply use your "Up, Down, Left, Right" key on the keyboard to move the ball.
  • The ball will not be able to pass the maze when they collide.
  • The only way to continue to the next level is reach the hole at the centre of the maze.

For Developers

The JavaScript files, game.js and phaser.io's game engine(phaser.min.js) are used to load and create images and game details into the emulator.

Game.js functions:

  • preload(): Loads images and all of the maze levels to the emulator.
  • create(): Starts and creates the game and identifies if it's going to the next level.
  • setup(): Initialises the game engine, map, tile layers and collision detection
  • destoryEverything(): Clears the game's map and tile layers for the next level of the game.
  • winLabel():To show the winning text.
  • update(): Updates the ball movement when arrow keys are being pressed. It also calls the checkOverLap function to check if the ball reaches the hole.
  • checkOverLap(): This function will called when the ball is overlapping the hole.
