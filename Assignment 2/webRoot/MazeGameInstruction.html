How to play

Simply use your "Up, Down, Left, Right" key on the key board to move the rectangle.
The rectangle will not be able to pass the black line.
The only way goes to next level is move the rectangle to the middle green hole.

Little bugs

Sometimes the picture of the maze game will not show up.
All you need to do is try run another game then back to run de maze game.
