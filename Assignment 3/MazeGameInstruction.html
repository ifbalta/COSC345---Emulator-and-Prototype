Maze Game Prototype Instruction Manual

  • Raeciel Reyes
  • Huijun Zhou

Dependencies

  • Phaser.io version 2.3.0

Requirements:

  • Install the emulator.
  • The game uses Phaser.io, and which requires a server to run. The emulator comes with a node server, or if you have python pre-installed, you can use python's SimpleHTTPServer.

How to play:

  • Simply use your "Up, Down, Left, Right" key on the keyboard to move the rectangle.
  • The ball will not be able to pass the maze when they collide.
  • The only way to continue to the next level is reach the black hole in the centre.

Bugs

  • Sometimes, there is a random collision between the ball and an empty space(which is a tile that we are unable to see)
    Solution: All you need to do is refresh the page and it will disappear

Reference

  • Phaser game engine for the game's collision detection and player's movement
